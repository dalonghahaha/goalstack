# 箩筐体育 1.0 版本路线图

## 项目概述

箩筐体育是一个面向足球爱好者的"数据优先、资讯辅助"平台，提供比赛查询、数据统计、球队/球员信息和资讯浏览功能。

## 技术栈

- 前端：Next.js 16.2.1 + React 19.2.4 + Tailwind CSS 4.2.2
- 后端：Next.js API 路由 + Prisma 7.6.0
- 数据库：PostgreSQL 16.2
- 数据可视化：Recharts
- 状态管理：Zustand
