# 箩筐体育 1.0 版本需求规格

**版本：** V1.0
**项目：** 箩筐体育
**日期：** 2026-03-29

---

## v1 需求（Active）

### 比赛中心（Home）

- [ ] **HOME-01**: 用户可以查看今日/明日/昨日比赛列表
- [ ] **HOME-02**: 用户可以按日期筛选比赛（昨天/今天/明天/日历选择）
- [ ] **HOME-03**: 用户可以按联赛筛选比赛（热门联赛 + 更多选项）
- [ ] **HOME-04**: 用户可以按状态筛选比赛（全部/未开赛/进行中/已结束）
- [ ] **HOME-05**: 用户可以点击比赛卡片进入比赛详情页
- [ ] **HOME-06**: 用户可以点击联赛名称进入赛事详情页
- [ ] **HOME-07**: 用户可以浏览右侧热门资讯并进入资讯详情页

### 比赛详情（Match）

- [ ] **MATCH-01**: 用户可以查看比赛基本信息（联赛、轮次、时间、场地、裁判）
- [ ] **MATCH-02**: 用户可以查看实时比分和比赛状态
- [ ] **MATCH-03**: 用户可以查看比赛详细统计数据（控球率、射门、射正、角球等）
- [ ] **MATCH-04**: 用户可以查看比赛事件时间轴（进球、黄牌、红牌、换人、点球）
- [ ] **MATCH-05**: 用户可以查看双方首发和替补阵容
- [ ] **MATCH-06**: 用户可以点击球员名进入球员详情页
- [ ] **MATCH-07**: 用户可以点击球队名进入球队详情页
- [ ] **MATCH-08**: 用户可以点击联赛名进入赛事详情页
- [ ] **MATCH-09**: 用户可以浏览比赛相关资讯

### 赛事详情（Competition）

- [ ] **COMP-01**: 用户可以查看赛事基本信息（名称、国家、赛季、参赛队伍）
- [ ] **COMP-02**: 用户可以切换赛季查看不同赛季数据
- [ ] **COMP-03**: 用户可以查看联赛积分榜（排名、球队、场次、胜平负、积分）
- [ ] **COMP-04**: 用户可以查看赛事赛程，按轮次筛选
- [ ] **COMP-05**: 用户可以查看射手榜和助攻榜
- [ ] **COMP-06**: 用户可以查看参赛球队列表并进入球队详情页
- [ ] **COMP-07**: 用户可以浏览赛事相关资讯
- [ ] **COMP-08**: 用户可以点击球队进入球队详情页

### 球队详情（Team）

- [ ] **TEAM-01**: 用户可以查看球队基本信息（名称、Logo、成立时间、主场、教练、所属联赛）
- [ ] **TEAM-02**: 用户可以查看球队阵容（按位置分类：门将、后卫、中场、前锋）
- [ ] **TEAM-03**: 用户可以点击球员进入球员详情页
- [ ] **TEAM-04**: 用户可以查看球队近期赛程（未来/过去）
- [ ] **TEAM-05**: 用户可以查看球队赛季数据统计（场次、胜平负、进球/失球、胜率）
- [ ] **TEAM-06**: 用户可以查看球队历史荣誉
- [ ] **TEAM-07**: 用户可以点击所属联赛进入赛事详情页
- [ ] **TEAM-08**: 用户可以点击某场比赛进入比赛详情页
- [ ] **TEAM-09**: 用户可以浏览球队相关新闻

### 球员详情（Player）

- [ ] **PLAYER-01**: 用户可以查看球员基本信息（姓名、头像、年龄、国籍、位置、所属球队、球衣号码）
- [ ] **PLAYER-02**: 用户可以切换赛季查看不同赛季数据
- [ ] **PLAYER-03**: 用户可以查看球员赛季数据（出场、进球、助攻、黄红牌等）
- [ ] **PLAYER-04**: 用户可以查看球员生涯荣誉
- [ ] **PLAYER-05**: 用户可以查看球员能力维度（雷达图或条形图）
- [ ] **PLAYER-06**: 用户可以点击所属球队进入球队详情页
- [ ] **PLAYER-07**: 用户可以浏览球员相关新闻

### 资讯中心（News）

- [ ] **NEWS-01**: 用户可以浏览资讯列表
- [ ] **NEWS-02**: 用户可以按资讯类型筛选（全部/比赛/赛事/球队/球员）
- [ ] **NEWS-03**: 用户可以按联赛筛选资讯
- [ ] **NEWS-04**: 用户可以按球队筛选资讯
- [ ] **NEWS-05**: 用户可以进入资讯详情页阅读正文
- [ ] **NEWS-06**: 用户可以点击资讯中的关联标签跳转到对应对象页
- [ ] **NEWS-07**: 用户可以浏览相关推荐资讯

### 通用功能（Common）

- [ ] **COMM-01**: 用户可以在页面间双向跳转（比赛↔赛事↔球队↔球员↔资讯）
- [ ] **COMM-02**: 用户可以切换明暗主题
- [ ] **COMM-03**: 用户可以在移动端获得良好的浏览体验
- [ ] **COMM-04**: 页面加载时显示骨架屏
- [ ] **COMM-05**: 无数据时显示空状态提示
- [ ] **COMM-06**: 接口异常时显示重试按钮
- [ ] **COMM-07**: PC 端支持分页，移动端支持加载更多

## v2 需求（Deferred）

以下功能是用户期望的，但 1.0 版本暂不实现：

- **关注功能**：用户可以关注联赛/球队/球员（P1）
- **收藏与历史**：用户可以收藏内容、查看浏览历史（P1）
- **全局搜索**：用户可以搜索比赛、赛事、球队、球员、资讯（P1）
- **开赛提醒**：比赛开始前提醒、比分变化推送（P1）
- **更多榜单维度**：球队数据、球员数据更多维度展示（P1）
- **图文直播**：比赛实时文字直播（P2）
- **视频集锦**：比赛精彩片段短视频（P2）
- **评论系统**：资讯评论、比赛评论（P2）
- **球员/球队对比**：数据对比功能（P2）
- **高级数据工具**：更丰富的数据可视化图表（P2）
- **分享功能**：内容分享到社交平台（P1）

---

## Out of Scope

以下功能明确不包含在 1.0 版本中：

- **用户登录注册** — 关注、收藏功能前置能力，1.0 暂不做
- **评论互动** — 社区风控成本高，需要配套审核系统，暂不做
- **赔率/博彩相关模块** — 与当前数据资讯平台定位冲突，存在合规风险
- **视频直播** — 版权和技术成本高，需要大量资源投入
- **社区帖子** — 需要大量运营和风控资源
- **即时推送** — 需要建立用户触达机制，P1 阶段实现
